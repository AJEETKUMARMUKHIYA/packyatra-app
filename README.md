PackYatra Application

PackYatra is a full‑stack travel management application built with a modular architecture:

Layer	Technology	Description
Frontend	React + Vite	Provides a responsive user interface for booking and managing travel packages.
Backend	ASP.NET Web API	Handles business logic, authentication, and communication between the UI and database.
Database	SQL Server	Stores application data, schema definitions, seed data, and stored procedures for efficient queries.
